ICT582 Lab 6:
Collections and Sequences

Learning Objectives:

  1. Understand the concept of collection and similarity and difference of common collection types such as tuple set, list and dictionary
  2. Understand and be able to use sets.
  3. Understand and be able to use dictionaries.
  4. Understand the similarity and difference between common sequence types such as string, tuple, list and range
  5. Be able to use dictionaries and other collection and sequence types to solve problems.

Required Reading:

  • Lecture Notes: Topic 6.

Reference:

  • https://www.python.org/about/gettingstarted/
  • ICT582 Lab Instructions

Required Software:

  • Python 3 and Idle 3

Exercises

  1. Revise Exercise 4 in Lab 5: this time, in addition to all other information, also ask the user to enter the name of the student. Save input for each student in a dictionary. At the end, display the information of all students, with each student in a separate line, in the format of 

    Name:  John Smith
StuentID:  1001
Q1:  60
Q2:  65
Final: 55
UnitMark: 59
Grade: P

  2. Expand Exercise 1 above. Show the following statistics:

    • Average, highest and lowest mark
    • The name and ID of all the students who got the highest grade
    • The name and ID of all the students who got the lowest grade
    • Total number of students failed in the unit (assuming 50% is the pass mark)

  3. Expand Exercise 6 in Lab 5: this time, the information for each student must be stored in a dictionary.

  4. Write a program that takes the lyrics of a song as a multi-line string and report the frequency of words used in the lyrics. Finally displays the list of words with the highest frequency and the list of words with the lowest frequency.

    Hints: refer to the example on words frequency in the lecture notes.
